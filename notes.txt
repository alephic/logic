Apply:

  func arg
  f x
  f x y
  f (g x) y

Symbolic constants behave as constructors when applied
Lambdas behave as functions when applied
Lambda:

  <arg> body
  <x> y
  <arg: constraint> body
  <x> <y> f x y
  <arg; corollary: constraint> body
  <x; y: c x y> y
  <arg; corollary1, corollary2: constraint> body
  <x; y, z: c x y z> y

Declare:

  {declaration} body
  {c x y} f x

Query:

  [name: constraint]
  [x: c x]
  [name; corollary: constraint]
  [name; corollary1, corollary2: constraint]

Fact table

  value at index -> sub-table or list of facts
